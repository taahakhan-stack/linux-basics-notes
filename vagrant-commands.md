# Vagrant Commands Cheat Sheet

## Initialization
vagrant init

## Start VM
vagrant up

## SSH into VM
vagrant ssh

## Stop VM
vagrant halt

## Restart VM
vagrant reload

## Destroy VM
vagrant destroy

## Check Status
vagrant status

## Suspend VM
vagrant suspend

## Resume VM
vagrant resume

## Re-provision VM
vagrant provision

## Show SSH config
vagrant ssh-config

## List boxes
vagrant box list

## Add box
vagrant box add ubuntu/focal64

## Remove box
vagrant box remove ubuntu/focal64
